# Universal Claude Prompt Templates

A curated, production-ready collection of prompt templates for Claude and other Large Language Models.

![Theme](https://img.shields.io/badge/Theme-Dark%20%2F%20Light-blueviolet)
![Responsive](https://img.shields.io/badge/Mobile-Ready-success)
![Hosting](https://img.shields.io/badge/Hosting-GitHub%20Pages-blue)

## Features

- **30+ Templates**: Optimized prompts for coding, analysis, creative writing, and more.
- **Copy-Paste Ready**: One-click copy functionality with visual feedback.
- **Favorites System**: Save your most-used templates (persisted via `localStorage`).
- **Dark & Light Mode**: Seamless theme switching.
- **Fast Search & Filter**: Find exactly what you need in seconds.
- **Fully Responsive**: Works perfectly on mobile, tablet, and desktop.

## Usage

1. Open `index.html` in any modern web browser.
2. Browse categories or search for specific keywords.
3. Click "Copy" to copy the template to your clipboard.
4. Replace the `[BRACKETS]` with your own data and use it with Claude.
